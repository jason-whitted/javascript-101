# Unary Operators: Typeof

> Determines the type of the operand.  Returns a string.

## Syntax
`typeof` operand

## Conversion
| Input        | Result     |
|-------------|-------------|
| `true`      | "boolean"   |
| `123`       | "number"    |
| "hello"     | "string"    |
| `undefined` | "undefined" |
| `null`      | "object"    |
