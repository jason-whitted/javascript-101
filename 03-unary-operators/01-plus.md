# Unary Operators: Plus

> Converts its operand to Number type.

## Syntax

`+`_operand_

## Conversion to Number
| Input       | Result |
|-------------|--------|
| `123`       | 123    |
| `NaN`       | NaN    |
| `"3.1415`"  | 3.1415 |
| `"xyz"`     | NaN    |
| `true`      | 1      |
| `false`     | 0      |
| `undefined` | NaN    |
| `null`      | 0      |
